# Voice-based-Email for visually impaired people
This is a final project Voice Based Email via Voice Commands for COM438.
1. Install following packages by using the pip command.
"speech_recognition
smtplib
BeautifulSoup
gtts
pyglet"
2. This project is going to use your microphone to recieve your commands.
3. It may read your mails, send mails, clear all your mails, clear some specific mails.
